# 🎬 Netflix Data Analysis Dashboard
## 📌 Project Overview

The **Netflix Data Analysis Dashboard** is an interactive data analytics project built using **Python, Pandas, NumPy, Matplotlib, Seaborn, and Streamlit**. It explores Netflix's catalog of movies and TV shows through interactive visualizations, dynamic filters, and insightful analytics.

The dashboard allows users to filter content by genre, type, country, rating, and release year while automatically updating key performance indicators (KPIs) and visualizations. The project demonstrates the complete data analysis workflow, including data cleaning, exploratory data analysis (EDA), visualization, and dashboard development.
## ✨ Features

* 📊 Interactive Streamlit dashboard
* 🎛️ Dynamic filtering by Genre, Type, Country, Rating, and Release Year
* 📈 Real-time KPI cards that update with applied filters
* 🎬 Movies vs TV Shows distribution analysis
* 🌍 Top Countries by Netflix content
* 🎭 Top Genres analysis
* ⭐ Ratings distribution
* 📅 Netflix content growth over the years
* ⏱️ Movie duration distribution
* 🗓️ Content added by month
* 🎥 Top Directors analysis
* 🎨 Custom dark-themed interface with responsive layout
## 🛠️ Technologies Used

## 📂 Dataset

This project uses the **Netflix Movies and TV Shows** dataset containing information about Netflix's content library, including movies and TV shows.

**Dataset includes:**

* Title
* Type (Movie / TV Show)
* Director
* Cast
* Country
* Release Year
* Date Added
* Rating
* Duration
* Genres

The dataset was cleaned and preprocessed before analysis by handling missing values, transforming columns, splitting multi-value fields, and preparing the data for visualization.

## 📈 Key Insights

* Movies make up a larger portion of Netflix's catalog than TV Shows.
* The United States contributes the highest number of titles on Netflix.
* Drama and International Movies are among the most popular genres.
* Netflix experienced significant content growth in recent years.
* TV-MA is one of the most common content ratings.
* Most movies have a runtime between 80 and 120 minutes.
* Content additions vary throughout the year, with certain months seeing higher activity.
* A small group of directors is responsible for multiple Netflix titles.
